Core Thesis

The traditional platform moat is being restructured — not eliminated. The question is not whether AI agents create switching costs, but what kinds, and how quickly they compound.

"The moat is time-compounded, feedback-reinforced depth. The new question is whether AgentMemoryDepth compounds faster than platform lock-in ever did."

Hermes Unbound · Chapter 2 · Working Draft

Key Constructs

AgentMemoryDepth — The compounding depth of agent-learned user context. Like data moats, but faster to build and more personal.
Pre-Articulation Capture — Agents that act before users articulate intent. The switching cost: losing context you didn't know you'd built.
Agent-Human Flywheel — Agents improve from human feedback; humans delegate more to agents. Each iteration deepens lock-in.
ADO — Agent Discoverability Optimization — The new SEO. How agents find, evaluate, and route to platforms on behalf of users.
AaaS vs SaaS — Agent-as-a-Service shifts the unit of value from subscriptions to delegated outcomes. Different switching dynamics entirely.
Agent-Human Switch — When an agent switches platforms on behalf of a user, the human never notices. Until they do.
